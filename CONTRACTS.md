# Contract Glossary

## Core
- **NFTMarketplace**: Main trading logic
- **BaseNFT**: ERC721 collection standard
- **MarketplaceToken**: Governance token

## Features
- **AuctionHouse**: English and Dutch auctions
- **OfferManager**: Bonding curve offers
- **RoyaltyRegistry**: Creator fee management

## Helpers
- **MarketplaceViewer**: Frontend data aggregation
- **TokenWhitelist**: Permitted currency list
- **CollectionManager**: Collection verification

## Libraries
- **MarketplaceMath**: Fee calcs
- **AuctionLogic**: Pricing curves
